Khám phá Domain, Hosting và URL

Domain là gì?

Domain (tên miền) là địa chỉ giúp người dùng truy cập website thay vì phải nhớ địa chỉ IP dài và khó nhớ.

Ví dụ: google.com, github.com, openai.com

Hosting là gì?

Hosting là nơi lưu trữ mã nguồn, hình ảnh, dữ liệu của website để website có thể hoạt động trên Internet.

Ví dụ: GitHub Pages, Vercel, Netlify, Hostinger...

URL là gì?

URL (Uniform Resource Locator) là địa chỉ đầy đủ của một tài nguyên trên Internet.

https://www.example.com/products/laptop?id=123

Cấu trúc URL

  • https → giao thức
  • www.example.com → domain
  • /products/laptop → đường dẫn
  • ?id=123 → query string

Ví dụ URL mẫu

URL 1

https://github.com/user/project

URL 2

https://www.youtube.com/watch?v=abc123
