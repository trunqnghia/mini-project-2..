Domain Hosting URL Ví dụ URL

Giới thiệu

Khi truy cập một website, chúng ta thường nhập địa chỉ như: https://www.google.com hoặc https://www.youtube.com.

Để website hoạt động cần có ba thành phần quan trọng: Domain, Hosting và URL.

1. Domain là gì?

Domain (tên miền) là địa chỉ dễ nhớ dùng để truy cập website. Thay vì phải nhớ một dãy số IP phức tạp, người dùng chỉ cần nhập tên miền.

Ví dụ Domain

  • google.com
  • youtube.com
  • github.com

Có thể xem domain giống như địa chỉ nhà giúp người khác tìm đến đúng nơi.

2. Hosting là gì?

Hosting là nơi lưu trữ toàn bộ dữ liệu của website như:

  • Trang HTML
  • Tệp CSS
  • JavaScript
  • Hình ảnh
  • Cơ sở dữ liệu

Nếu Domain là địa chỉ nhà thì Hosting chính là ngôi nhà chứa toàn bộ nội dung của website.

3. URL là gì?

URL (Uniform Resource Locator) là địa chỉ đầy đủ dùng để truy cập một tài nguyên trên Internet.

https://www.example.com/products/laptop?id=10

Cấu trúc URL

Thành phần Ví dụ Ý nghĩa
Protocol https:// Giao thức truyền dữ liệu
Domain example.com Tên miền website
Path /products Đường dẫn đến tài nguyên
Query ?id=10 Tham số gửi lên máy chủ

4. Minh họa URL mẫu

Ví dụ 1

https://www.google.com/search?q=html
  • Protocol: https
  • Domain: google.com
  • Path: /search
  • Query: q=html

Ví dụ 2

https://github.com/user/project
  • Protocol: https
  • Domain: github.com
  • Path: /user/project
  • Không có Query String

Tóm tắt

Khái niệm Chức năng
Domain Địa chỉ của website
Hosting Lưu trữ dữ liệu website
URL Địa chỉ đầy đủ của tài nguyên
